# 🧵 ThreadCraft Textiles - IBM Cognos Analytics Dashboard

## Overview
Enterprise-grade production monitoring and business intelligence dashboard for textile manufacturing operations. Integrated with IBM Cognos Analytics for real-time operational insights, export compliance tracking, and sales performance analytics.

## Features
- **🔐 Secure Authentication** - Role-based login system
- **📊 Live KPI Dashboard** - Real-time production metrics (efficiency, waste, fulfillment)
- **📈 IBM Cognos Integration** - Embedded operations dashboard with full interactivity
- **🎨 Interactive Charts** - Revenue by garment type, production trends, waste analysis
- **⚠️ Alert System** - Orders below 80% fulfillment tracking
- **📱 Responsive Design** - Works on desktop, tablet, and mobile devices

## Tech Stack
- HTML5 / CSS3 / JavaScript (Vanilla)
- Chart.js for data visualization
- IBM Cognos Analytics Embedded Framework
- Font Awesome Icons
- Google Fonts (Poppins)

## Key Metrics Tracked
- Total Units Produced: 284,500+
- Production Efficiency: 86.3%
- Fabric Waste: 4.2%
- Export Fulfillment: 91.2%
